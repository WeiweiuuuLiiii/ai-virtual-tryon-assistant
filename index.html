AI Try-On
Workflow Features Roadmap

Pre-Launch AI Product MVP

Virtual try-on and outfit decision support for real shoppers.

Create a personal fitting model, upload clothing items, preview outfits, and receive context-aware buy-or-skip recommendations.

Core Workflow

  1. User profile / body model creation
  2. Garment image upload
  3. Outfit composition
  4. Scene and context selection
  5. AI fit, style, weather, and occasion analysis
  6. Buy-or-skip recommendation

Personal Fitting Model

Uses measurements or full-body input to create a user-centered try-on profile.

Context-Aware Analysis

Considers weather, travel, occasion, comfort, and repeat-use likelihood.

Buy-or-Skip Decision

Explains fit risks, style match, practicality, and whether the item is worth buying.

Current Status

Work in progress. Product architecture, core workflow, and first frontend MVP structure are being developed.

AI is used as an engineering partner, not a replacement for human judgment.
